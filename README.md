# Configuration

## Travis

Configure environment variables

```
DOCKER_USERNAME myusername
DOCKER_PASSWORD secretsecret
```

Setup github integration

## Docker cloud

Configure environment variables

```
POSTGRES_USER
POSTGRES_PASSWORD
POSTGRES_HOST
SECRET_KEY_BASE
```
